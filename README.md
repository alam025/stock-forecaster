## 🚀 About

> **Built ensemble model achieving 4.2% MAPE on 5-day ahead S&P 500 forecasts**

**Stock Price Forecaster** is a production-ready **Data Science** project built with **Python, TensorFlow, Prophet, Plotly**.
Deployed on **Streamlit Cloud** with full CI/CD pipeline.

```bash
# 1. Clone

cd stock-forecaster

# 2. Install
pip install -r requirements.txt

# 3. Run
python main.py
```
